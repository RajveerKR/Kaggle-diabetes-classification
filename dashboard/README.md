## Diabetes Risk Analytics Dashboard

This dashboard presents population-level insights derived from a machine learning model trained on the Kaggle Diabetes Prediction dataset.

### Overview
- Total individuals analysed: 300,000
- Risk bands: High, Medium, Low
- KPIs include population counts and high-risk proportion

### Purpose
The dashboard is designed to support analytical understanding and planning, demonstrating how predictive model outputs can be translated into accessible insights for decision-making.

### Ethical note
The outputs shown are model-predicted risk probabilities and do not represent clinical diagnoses. Results should be interpreted for analytical and planning purposes only.
